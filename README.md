# KYC verification — German entities

A walkthrough of two Vicoland × MangoPay KYC verification flows, built from screen recordings:

1. **German sole trader / freelancer** — the lighter flow (ID + proof of registration), with the AI-check consent step and document-mismatch detection shown end to end.
2. **German business** — the fuller company flow (register extract, articles of association, beneficial-owner declaration), with AI document checks.

Each flow shows one screenshot per step with an explanation of what it does and how it's customised for German entities. Click any screen to open it full-screen and step through the flow.

**Live page:** https://ivanvuksanov.github.io/kyc-german-flows/

Static site — open `index.html` locally, or view via GitHub Pages.
